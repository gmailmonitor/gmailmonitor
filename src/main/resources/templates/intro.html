3rd-Party APIs

OAuth 2.0 with Google APIs - The user authenticates with Google's servers, granting an access token to this application. This application neither handles nor stores the user's account credentials.
Gmail API is called in order to obtain basic statistical information on the user's Inbox
Google Cloud Pub/Sub API is leveraged in order to obtain realtime notification of user's mailbox updates
Google API Client Library for Java performs HTTP requests against Google REST APIs


Persistence

ORMLite eliminates the need to write most SQL queries as String objects, while providing Object-Relational Mapping of database entities to Java classes.
H2 Database Engine - a lightweight SQL database engine, easily meets the storage needs of this application with minimal added complexity


Front End

D3.js(Data Driven Documents) to present data in chart format
Thymeleaf within Spring MVC Spring Boot with embedded Tomcat


Front End

jackson-databind by FasterXML for easier translation between JSON and Java classes.




Database schema updates are implemented programmatically. On startup, the database schema is automatically updated to the latest version. This allows the application to be started with any version of the database. No scripts to run manually. Externalized configuration using property files
